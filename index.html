许巍是一种瘾

Apr 17th, 2016 9:37 pm

“许巍是一种瘾"，从本科起听过的一句话，但只知道许巍是一个长得比较丑的歌手。后来听了《蓝莲花》，觉得他的声音有一种沧桑感，很有特色，这也许是摇滚乐中比较重要都元素，比如汪峰。说不上迷恋，但挺喜欢歌声中的自由感。至于“许巍是一种瘾”，完全只是一句觉得耳熟能详的话而已。

今天偶然再次听到了《蓝莲花》，已不是当年都感觉，只是还是喜欢。就顺带把许巍都歌都找出来听了一遍，歌声中的自由、孤独、却又真诚、温暖，感动着我。终于有些明白，原来“许巍是一种瘾”是这种感觉。大多数人都生活似乎都不尽人意，除了物质都因素，还有自我认知、社会认知，自我价值和社会价值的实现。有时候很多东西是冲突的，自由和梦想，总是与现实生活格格不入，幸福感的降低，大多源于自我不能得到满足。

这个时候，许巍带着他都歌声出现了。得到了很多人的响应和共鸣，从歌声中听出了郁闷、彷徨、孤独、自由，最终却是温暖和感动的力量。

许巍是一种瘾，源于理想和自由与现实的碰撞。热爱生活，是对这种碰撞最好的回应。

APPT2015小记

Sep 2nd, 2015 8:18 am

        APPT(高级计算机并行处理技术研讨会)每两年一次，且国内国外交叉举办，对于仅有的两年半研究生光阴而言，能参加一次，也算很幸运。本次会议，由中国计算机学会体系结构专委会和浪潮集团承办，在济南高新区的喜来登酒店进行。比之于上次参加Chinasys的紧张，团体5人组的小分队，定下的是轻松愉悦的基调。比较遗憾的是，参会的很多都是体系结构届大牛，跟我等小喽啰，基本没有交流。废话不多说，还是记录下本次会议，自己的收获吧。
        会议安排为8月20日，8月21日两天。按照惯例，第一天上午，体系结构届大佬做一些开场性的报告：

中科院沈绪榜院士，芯片计算机体系结构的思考

主要从数学思想，图灵机等的发展，提出芯片设计技术可以从可计算数的表示，可计算机数的操作和可计算性的操作三个方面，消除高级语言和汇编语言中不符合数学语言的概念和表示方法的内容，来提升芯片性能。

北航钱德沛教授，中国高技术计划高性能计算发展的现状与未来

主要从近年来高性能计算的发展以及国家政策上，提出了高性能计算发展未来的三个方向：一，E级高性能计算系统研制；二，高性能应用软件开发；三，高性能计算环境的研发。

马里兰大学屈钢教授，Hardware in Cybersecurity:from the Weakest Link to Great Promises

主要介绍硬件安全性的问题，因为没有做过这一块的，听不是太懂。只记得一个叫Cipher的工具，还有几个术语： TPM：Trust Platform Module，可信平台模块（Trusted Platform Module）是一种植于计算机内部为计算机提供可信根的芯片。该芯片的规格由可信计算组（Trusted Computing Group）来制定。符合TPM(可信赖平台模块)标准的安全芯片，它能有效地保护PC、防止非法用户访问。
PUF： Physical Unclonable Function. 不可复制功能 (PUF) 技术可用于实现标签与阅读器的相互认证。PUF 技术采用硅片独特的物理特性和 IC 制造过程的变异性来识别各个硅芯片，判断它们的真伪性，无需采用密钥或存储密钥。 安全和认证方案提供商 Verayo 获此专利。

陈继承，高效能服务器和存储技术国家重点实验室

主要介绍了高效能服务器的一些技术，浪潮的Inspur Tiansuo K1。

8月20日下午，一些会议收录的论文和专家报告。分为两个会场，对于有选择综合症的人来说，想挑着听报告都有点麻烦。

Improving Memory Access Performance of In-Memory Key-Value Store using Data Prefetching Techniques

设计提出一种IMkvs的快速预取技术，解决大数据问题。
关键设计技术：Memory Controller：IRR，PACU，PSCU.

DDC：Distributed Data Collection Framework for Failure Prediction in Tianhe Supercomputers 国防科大

分布式数据收集技术。
关键设计技术：DDC Collector Manager: ES collector, RS collector, FS collector.

Optimizing the MapReduce Framework for CPU-MIC Heterogeneous Cluster 国防科大

主要介绍CPU-MIC 异构集群下优化的MapReduce框架。
关键技术：在MapReduce System与MIC之间设计Runtime Library，MIC Memory Management采用Array Buffer和Combine Buffer两种模式。SIMD friendly map(向量友好map)和Pipeline Reduce。

Stable Matching Scheduler for Single-ISA Heterogeneous Multi-core Processor 中科院

在异构多核处理器上采用stable matching scheduler 实现性能优化。
特点：将经济学stable matching 应用于异构多核处理器，实现了性能提升。

Neural Network Processor 中科院

将人工神经网络应用于处理器设计。
成果：DianNao : 采用硬件运算单元的分时复用技术，将硬件运算单元与算法神经元对应。小尺度但大规模支持神经网络。
        DaDianNao:多神经网络处理器。
        PuDianNao:针对不同机器学习算法的人工神经网络处理器。
        ShiDiannao:图像处理的人工神经网络处理器。

Reproducing Concurrency Bugs Using Local Clocks 中科院

用本地时钟重现并行bug。主要解决三个问题：时钟差值d，out-of-order 执行，本地时钟overflow.这篇论文可作为文献参考，仔细阅读。

Energy Efficient Router: Architecture, Prototype and Algorithms 北理工

路由器通过感知网络流量，自动调节各芯片工作，从能提升路由器性能。

8月21日上午，专家报告。

认知计算 秦勇，IBM中国研究院资深科学家

介绍计算机发展，从制表时代，编程时代，到认知计算。 认知计算研究的发展趋势：
Passive to Active,
Supervised to Unsupervised,
Natural Language Text to Audio,Image,Video,other media,
Captive to Ubiquitious
Scalability low to high

New IT for Big Data and Mobile Cloud Era 陈宏，VMare研发总监

大数据时代——并行数据库技术 宋一平，SAP

主要介绍并行数据库技术。 涉及关键技术：
多处理器/多节点网络互连技术
存储访问体系结构
分布式数据与文件管理
并行计算的任务划分和算法设计
并行程序设计模型和语言及实现。
并行数据库与分布式数据库的区别与联系：
相同点：
网络连接各个处理节点
所有节点构成一个逻辑整体
各节点数据访问透明等
不同点：

并行数据库分布式数据库
应用目标    提高数据库的性能和可用性    实现场地自治和数据的全局透明共享
实现方式    结点间采用高速网络连接    结点间一般采用局域网或广域网相连
结点地位    结点非独立    高度自治性


        8月21日下午，学术界与工业界研讨会。之前我一直认为将学术研究应用于实践才有价值，不能基于实际问题的研究也只能是空中楼阁。这次有了政府人员的参与，感受就与Chinasys不同，国家的高度，学术界和工业界本应该各司其职，就是研究的人就应该研究没有的东西，工业的人就该搞实际的量产。尽管如此，工业界与学术界的沟通仍然是一个很大的问题。当然，作为小硕一枚，还是觉得应该接地气的好，脚踏实地做好该做的研究，再说高屋建瓴的事。
        接下来，去参观了济南云计算中心，是个更轻松的环节。美女姐姐仔细介绍了浪潮的各种高效能服务器的研究，平台架构，操作系统，应用平台等技术以及其在各行各业的应用。最后的答谢晚宴，APPT2015圆满结束。
